# LESSONS — Domus flight test

1. **Brand before aesthetic fantasy** — Real logo multi-color D + orange CTA beats invented atelier bone/bronze.
2. **Local-business flight tests need Local SEO knowledge + WhatsApp**, not just Premium-Hero.
3. **Gap honesty surfaces ROI** — Missing Next lead adapter and SEO-Foundation Active are the highest-value follow-ups for DevOS 1.8+.
4. **Live site skim** (Firecrawl) grounds IA: quote fields, FAQ, multi-island phones, off-site blog opportunity.
5. **`NEXT_PUBLIC_SITE_URL` / `metadataBase` must fail closed in production** — defaulting to `http://localhost:3000` poisons canonicals, sitemap, and robots (audit 2026-07-19).
6. **Optional webhook ≠ durable leads** — accepting `{ok:true}` when nothing persisted is a silent production incident.
7. **In-memory rate limits are not Secure-API** on serverless — document as WARNING, ship Redis/Upstash (or equivalent) for launch.
8. **Never invent testimonials** — empty trust section or verified quotes only; fabricated names are brand/legal risk.
9. **Stock photography is not a manufacturer portfolio** — Unsplash reads as template; kill launch readiness.
10. **Contract tests must import the real schema** — duplicated zod in `scripts/lead.contract.test.mjs` drifts from `src/lib/lead.ts`.
